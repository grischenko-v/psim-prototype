# PSIM Prototype

Interactive UX prototype for a Physical Security Information Management operator workspace.

## Screens

- Monitoring queue with incident search and filters
- Incident workspace with SOP confirmation, comments, linked events, map, and video context
- Event workbench with user-created filtered event tables
- Incident archive and reporting filters
- Map view
- VMS camera catalog and quadrator
- Supervisor dashboard

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

GitHub Pages deployment is configured through `.github/workflows/pages.yml`.
